WPEssential Elementor Blocks is a flexible extension of WordPress. It is open-source page builder solution in WordPress. Create any theme, anywhere and make your way. There is no `PHP`, `CSS`, `HTML` and `Javascript` coding knowledge need.


== Description ==

WordPress plugin for easy theme creation. No coding required. Open-source with page builder. Build themes hassle-free.


== Supported Plugins ==

== Installation ==
= Manual installation =
Manual installation method requires downloading the WPEssential plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
